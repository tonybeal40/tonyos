# Objection Handling

## "We need to think about it."
“Totally fair. What part feels unclear—timeline, ROI, or implementation effort?”

## "Too expensive."
“Understood. Compared to lost deals from weak follow-up, this is usually the cheaper path. Want a phased option?”

## "We’re busy right now."
“That’s exactly why this helps. It removes pipeline chaos so the team executes faster with less context switching.”

## "Can you prove it?"
“Yes. I’ll show the workflow and outcomes from v0.2. We can start with a scoped pilot and clear deliverables.”
