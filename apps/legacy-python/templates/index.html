From there, Tony maps targeting, outreach, and content around real buyers, real titles, and real regions so execution is simple.

Setup Instructions

Data Sync Info

Auto-sync is always on. Every time you save in any app, your data automatically saves to your local database AND syncs to Google Sheets. You never need to manually update files.

Connect Google Sheets

  1. Create a Google Sheet
  2. Extensions > Apps Script
  3. Paste code from docs/apps-script-endpoint.js
  4. Deploy > New deployment > Web app
  5. Click the indicator (bottom-right) and paste URL

Run as Desktop App

  1. Download zip from Replit (... menu > Download)
  2. Extract the folder
  3. Open terminal in electron/ folder
  4. Run: npm install
  5. Run: npm start

What Gets Synced

Each app syncs to its own tab in your Google Sheet:
Journal - entries, mood, gratitude
CRM_Contacts - your connections
CRM_Activity - outreach logs
Budget - transactions
Daily - daily focus entries
Jobs - job tracking
