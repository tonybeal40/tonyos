What This App Will Do (Locked and Simple)

App name: tonyrevops-internal-downloads

Purpose:
A neutral internal page with:

Two sections

Downloads

Internal

Perfect spacing

Centered layout

Upload once, forget forever

Supports PDF or DOCX download

No analytics
No branding
No selling
No job-seeking signal

Layout (Final, Don’t Change Later)

Centered card with three fixed slots (this is important so it never looks empty or crowded):

LinkedIn Background PDF

Resume / One-Pager

Internal Notes or Reference Doc

You can swap the files later without touching layout or code structure.

Folder Structure (Replit)
tonyrevops-downloads/
│
├── index.html
├── style.css
├── app.js
└── files/
    ├── linkedin-background.pdf
    ├── resume.pdf
    └── internal-notes.pdf


You only replace files inside /files/.
