System Description - Tool Management II

Tool Management II (TM-II) is designed for tablet manufacturers. The program provides tablet manufacturers with tool control and inspection features to improve product quality and consistency, eliminate waste, and also help ensure compliance with cGMP record keeping requirements.

Core Functions

  • Tool storage and location tracking
  • Inspection and tolerance reporting
  • Usage monitoring and tablet count tracking
  • Tool drawing file management
  • Report generation (maintenance, usage, summary)
  • Purchase order creation for tooling
  • Integration with Natoli Laser Vision System (LVS)

System Architecture

Tool Directory → Main table containing entries for each tool (punches, dies)

Sets Directory → Links tools to sets for production

Products Directory → Contains entries for every product/tablet with default limits
